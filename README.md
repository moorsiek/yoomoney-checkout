Easy and complete YooKassa (formerly Yandex Checkout) integration for Laravel

## Requirements

* PHP **7.3** or above
* Laravel **6** or above 

## Installation

You can install the package via composer:

```bash
composer require orkhanahmadov/yandex-checkout
```
